\paragraph{Options}
The options for Neighbors are shown in Table \ref{table:hc_opt_neighbors}. The option values correspond to distances in feet to the nearest neighbor on the left and right sides of the building. The options set modeling properties corresponding to the distance of shading objects, while the shading object height is assumed to be the same as the housing unit's height. It's also assumed that there are no neighbors to the front and back of the unit. The None option means there are no neighbors.
