\paragraph{Description}
% \input{characteristics/description/Vacancy Status}
The vacancy status (occupied or vacant) of the housing unit.

\paragraph{Distribution Data Sources}
% \input{characteristics/sources/Vacancy Status}
\begin{itemize}
\item
  2019 5-year PUMS from the University of Minnesota.
\end{itemize}

\paragraph{Direct Conditional Dependencies}
\input{characteristics/dependencies/Vacancy Status}

\paragraph{Options}
The Vacancy Status options are either ``Occupied'' or ``Vacant''. Vacant units are assumed to be vacant for the full calendar year.

\paragraph{Distribution Assumptions}
% \input{characteristics/assumptions/Vacancy Status}
\begin{itemize}
\item
  Where sample counts are less than 10 (434 / 11,680 rows), the State
  average distribution has been inserted. `Mobile
  Home' does not exist in D.C. and is replaced by
  `Single-Family Detached.'
\end{itemize}
