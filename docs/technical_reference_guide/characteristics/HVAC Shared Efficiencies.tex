\paragraph{Options}
Shared HVAC system options are shown in Table FIXME. 89\% of homes have no shared system.

For homes with only shared cooling (and in-unit heating or no heating), ResStock currently approximates these fan coil systems as a mini-split heat pumps.

For homes with only shared heating (and in-unit cooling or no cooling), ResStock models two shared boilers with different fuel types (note: the heating system fuel is set by the Heating Fuel characteristic, which is why it's not listed here).

For homes with both shared heating and cooling, the options are a combination of the options for shared cooling only and shared heating only.
