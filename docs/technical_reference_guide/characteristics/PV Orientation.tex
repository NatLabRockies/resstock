\paragraph{Description}
% \input{characteristics/description/PV Orientation}
The orientation of the PV system.

\paragraph{Distribution Data Sources}
% \input{characteristics/sources/PV Orientation}
Constructed using LBNL's 2020 Tracking the Sun report \citep{LBNLTTS2019}.

\paragraph{Direct Conditional Dependencies}
\input{characteristics/dependencies/PV Orientation}

\paragraph{Options}
The options for PV orientation are the cardinal and subcardinal directions as shown in Table \ref{table:hc_opt_pv_orientation}. The ``None'' option is used for housing units that do not have PV systems.

\input{characteristics/options/PV Orientation}

Property definitions are shown in Table \ref{table:hc_arg_def_pv_orientation}. See the OpenStudio-HPXML \href{https://openstudio-hpxml.readthedocs.io/en/v1.10.0/workflow_inputs.html#hpxml-photovoltaics}{Photovoltaics} documentation for additional information.

\input{characteristics/properties/PV Orientation}

\paragraph{Distribution Assumptions}
% \input{characteristics/assumptions/PV Orientation}
\begin{itemize}
    \item PV orientation mapped based on the azimuth angle of the primary array (180\degree~is south-facing).
    \item The orientation is not aligned with the roof deck's normal directions from the Orientation characteristic (Section \ref{sec:orientation}).
\end{itemize}
