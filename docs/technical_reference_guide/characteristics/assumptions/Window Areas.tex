\begin{itemize}
\item The window to wall ratios (WWR) are exponential weibull distributed.
\item Multi-Family with 2-4 Units distributions are independent of Geometry Stories
\item Multi-Family with 5+ Units distributions are grouped by 1-3 stories, 4-7 stories, and 8+ stories
\item High-rise Multi-family buildings (8+ stories) have a 30\% window to wall ratio (WWR)
\item SFD, SFA, and Mobile Homes are represented by the SFD window area distribution
\end{itemize}
