\begin{itemize}
\item RECS 2020 has Multi-Family 5+ units marked as Not Applicable. These units are replaced by data from the No Place Like Home Study (\href{https://www.nrel.gov/docs/fy22osti/81065.pdf}{https://www.nrel.gov/docs/fy22osti/81065.pdf}) Table 1 Scenario 2. Multi-Family 5+ units owners is assumed for 12\% units having electrical access. This fraction is based on the average of mid-capacity apt and high-capacity apt. Multi-Family 5+ units owners is assumed for 28\% units having electrical access.
\item Units reported to have an L1 or L2 charger in the field EVCHRGTYPE are assumed to have outlet access.
\item L1 chargers without outlet access are units with EVs, but report no outlet access within 20 ft of vehicle parking.
\item Due to low sample count, the tsv is constructed by downscaling a dwelling unit sub-tsv with a household sub-tsv. The sub-tsvs have the following dependencies:
\item Dwelling unit sub-tsv : deps=['Geometry Building Type RECS', 'Vintage ACS', 'Electric Vehicle Charger'] with the following fallback coarsening order
  \begin{itemize}
  \item Vintage ACS to the National distribution
  \item Combining Geometry Building Type RECS together into SFD+MF+SFA and MF 2-4+MF 5+ bins
  \end{itemize}
\item Household sub-tsv : deps=['Geometry Building Type RECS', 'State' 'Tenure', 'Federal Poverty Level'] In combining the dwelling unit sub-tsv and household sub-tsv, the conditional relationships are ignored across (['Vintage ACS', 'Electric Vehicle Charger'], ['Tenure', 'Federal Poverty Level']).
\end{itemize}
