\begin{itemize}
\item 2020 Decennial Redistricting data was used to map tract level unit counts to census blocks.
\item 1,099 cities are tagged in ResStock, but there are over 29,000 Places in the Census data.
\item The threshold for including a Census Place in the City.tsv is 15,000 dwelling units.
\item The value 'In another census Place' designates the fraction of dwelling units in a Census Place with fewer total dwelling units than the threshold.
\item The value 'Not in a census Place' designates the fraction of dwelling units not in a Census Place according to the 2010 Census.
\end{itemize}
