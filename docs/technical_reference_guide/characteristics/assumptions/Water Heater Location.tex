  \item 3 + Garage combined.
  \item Single-/Multi-Family + Foundation combined + Attic combined + Garage combined.
  \item 5 + pre-1960 combined.
  \item 5 + pre-1960 combined / post-2020 combined.
  \item 7 + IECC Climate Zone lumped into: 1-2+3A, 3B-3C, 4, 5, 6, 7 except AK, 7AK-8AK.
  \item 7 + IECC Climate Zone lumped into: 1-2-3, 4-8.
