\paragraph{Options}
Homes can be assigned to have shared cooling, shared heating, shared both heating and cooling, or no shared systems. No modeling properties are assigned based on this characteristic, but these options are used as dependencies in other ResStock characteristics, such as HVAC Shared Efficiencies, where properties are assigned.
