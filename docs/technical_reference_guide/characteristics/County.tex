\paragraph{Options}
The County characteristic options are structured using the State name and County name. An example of the option corresponding to Denver County in Colorado would be ``CO, Denver County''. Option properties are shown in Table \ref{table:hc_arg_def_county}.

For each county, a single zip code is assigned.

EnergyPlus weather files are assigned as ``../../../weather/<County GISJOIN>.epw'' where <County GISJOIN> is the NHGIS GISJOIN value for the county. The NHGISJOIN for a county always starts with the letter ``G'' followed by the state's two-digit FIPS, a ``0,'' and then the County 3-digit FIPS code. An example of this property assignment for the option ``CO, Denver County'' is ``../../../weather/G0800310.epw''.
