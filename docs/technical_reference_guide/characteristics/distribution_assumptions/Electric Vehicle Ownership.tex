\item Due to low sample sizes, fallback rules applied with lumping of:
  \begin{itemize}
  \item [1] Federal Poverty Level lumped every 100\%
  \item [2] Federal Poverty Level lumped every 200\%
  \item [3] Building type is consolidated into three bins 1) single-family, 2) multi-family, and 3) mobile home bins
