\paragraph{Distribution Assumptions}
% \input{characteristics/assumptions/Lighting}
\begin{itemize}
 
\item
  Qualitative lamp type fractions in each household surveyed are
  distributed to three options representing 100\% incandescent, 100\%
  CFl, and 100\% LED lamp type options.
\item
Qualitative portion of inside light bulbs in 2020 RECS microdata is 
mapped to quantative percentage as: 
None: 0\%; Some: 20\%; About half: 50\%; Most: 80\%; All: 100\%. 
Then the sum of three types of lighting options is normalized to 100\%.
\end{itemize}
