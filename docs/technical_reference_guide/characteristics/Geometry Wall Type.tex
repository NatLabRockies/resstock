\paragraph{Options}
ResStock models different wall types as shown in Table \ref{table:hc_opt_geometry_wall_type}. This is the interior structure of the wall, not the exterior cladding. This characteristic does not have any modeling properties, but other characteristics that influence the thermal properties of the model are dependent on Geometry Wall Type.
