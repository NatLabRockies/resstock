\paragraph{Description}
% \input{characteristics/description/Geometry Story Bin}
Tags the building in which the housing unit is located as having more than 8 stories or less than 8 stories.

\paragraph{Distribution Data Sources}
% \input{characteristics/sources/Geometry Story Bin}
\begin{itemize}
    \item U.S.~EIA 2009 RECS microdata.
\end{itemize}

\paragraph{Direct Conditional Dependencies}
\input{characteristics/dependencies/Geometry Story Bin}

\paragraph{Options}
The options are either <8 stories or 8+ stories. This characteristic is an aggregation of the Geometry stories characteristic to identify units in high-rise buildings. No properties are assigned based on this housing characteristic, but it is used as a dependency for other input files.

\input{characteristics/options/Geometry Story Bin}

\paragraph{Distribution Assumptions}
% \input{characteristics/assumptions/Geometry Story Bin}
\begin{itemize}
    \item The probability values are a direct mapping of the Geometry Stories characteristic.
\end{itemize}
