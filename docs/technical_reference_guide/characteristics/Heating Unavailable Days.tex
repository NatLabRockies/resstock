Number of days in a year the heating system is unavailable.

\paragraph{Distribution Data Sources}

\begin{itemize}
	\item U.S. EIA 2020 Residential Energy Consumption Survey (RECS) microdata.
\end{itemize}

\paragraph{Distribution Assumptions}
% \input{characteristics/assumptions/Heating Unavailable Days}
\begin{itemize}
	\item Where samples are less than 10, the data is aggregated in the following order until there are no rows with less than 10 samples:
	\begin{itemize}
		\item The Federal Poverty Level dependency is aggregated every 100\% bins
		\item The Federal Poverty Level dependency is aggregated every 200\% bins
		\item The Geometry Building Type RECS dependency is aggregated into single-family, multi-family, and mobile home bins.
		\item The Cooling Unavailable Days dependency is aggregated into days, weeks, month, and year round bins.
		\item The Cooling Unavailable Days dependency is removed.
	\end{itemize}
\end{itemize}
