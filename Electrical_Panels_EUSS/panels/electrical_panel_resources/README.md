# Electrical Panel Resources
This folder contains resource files created for assigning electrical panel service ratings and available breaker spaces using ResStockArguments.

These files are used by `../panel_capacity_break_space_prediction_based_on_resource_files_resstock3_2_0.py` to assign panel characteristics to a given buildstock.csv, results_up00, or OEDI-formatted result file.

These are copied from resstock-estimation here:
https://github.com/NREL/resstock-estimation/tree/develop/resstock/measures/ResStockArguments/resources/electrical_panel

The file creation scripts along with a README on sources are available in resstock-estimation here: 
https://github.com/NREL/resstock-estimation/tree/develop/sources/electrical_panel
